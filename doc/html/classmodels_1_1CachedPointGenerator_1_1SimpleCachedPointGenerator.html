Detailed Description

Простой кэширующий генератор.

В первый раз сохраняет результат работы point_generator при scale = 1. В последующие вызовы координаты точки умнажаются на scale, без повторной генерации

def func(x: float) -> POINT:
return x*x
gen = FuncPointGenerator(func)
gen = SimpleCachedPointGenerator(gen)
for point in gen(((0, 100), (0, 100)), 1):
print(point)
